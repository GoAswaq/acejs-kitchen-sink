There are two types of masks:

  • Masks that block the access to an ACE field.
  • Masks that block the access to the entire screen.

Each ace element when it's performing an ajax call will block itselef with a mask. But these masks may also be triggered programatically, like in the examples bellow.

Ex. 1 - Screen Mask

  • The screen mask prevents any user access to the page elements.
  • Optionally a text, and / or a loading animation might be presented.
  • After the screen mask is displayed, it will hide itself after 3 seconds

Ex. 2 - Ace Elements Masks

  • Each element triggers its mask when performing an ajax call.
  • The masks can also be triggered programaticaly, like in this example.
