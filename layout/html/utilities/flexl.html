CSS Flexbox Layout makes it easier to design flexible responsive layout structure without using float or positioning.

Ex. 1 - Flex Layout System

  • ACE comes with a set of classes which are wrappers over various flex layout settings.
  • There are a set of classes that apply on the container in which the flexible elements find themsleves, and another set of classes that apply on the flexible elements themselevs:
  • Properties of the container:
    • direction: ace-flex-column, ace-flex-row,ace-flex-column-reverse, ace-flex-row-reverse
    • wrap: ace-flex-wrap-no, ace-flex-wrap-yes, ace-flex-wrap-yes-reverse
    • justify content: ace-flex-justify-end, ace-flex-justify-start, ace-flex-justify-center, ace-flex-justify-spacebetween, ace-flex-justify-spacearound.
  • Properties of the elements in the container:
    • align self: ace-flex-align-self-strech, ace-flex-align-self-center, ace-flex-align-self-start, ace-flex-align-self-end. This OVERRIDES the align items property of the container.
    • flex-grow: ace-flex-1, ... to ace-flex-12, ace-flex-20
    • flex-shrink: ace-flex-shrink-1, ... to ace-flex-shrink-10
  • Attention, in the example bellow:
    • allign self classes are applied only on the elements on even numbered positions: 2,4,6,8,10.
    • flex grow and flex shrink classes, apply 1 to each of the 10 elements inside, from 1 to 10.
