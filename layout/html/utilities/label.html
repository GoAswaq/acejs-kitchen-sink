Labels are not stand alone elements.

They could be considered pseudo-elements, which are used by other ACE Fields.

Ex. 1 - Label Modifier Classes

  • These classes modify the appearance or functionality of labels.
  • The first section of ace elements, marked with none have no custom classes applied, and no explicit label length set.
  • On all the other sections, the label length has been set to 100px.
  • The modifier classes are:
    • ace-no-label: hides the labels.
    • ace-label-width-auto: the explicit label length will be ignored.
    • ace-label-max: the explicit label length will be ignored, and the label will stretch so that the entire text will be visible.
    • ace-label-text-left: forces the text in the label to be aligned left
    • ace-label-text-right: forces the text in the label to be aligned right
