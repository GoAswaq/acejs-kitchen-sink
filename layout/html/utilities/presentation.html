One of the types of panels predefined in the ace classes.

Presentation Panels have 3 main divs:

      a container, identified by the ace-presentation-container class
      a main child of the container parent identified by the class ace-presentation-main
      a smaller child of the container parent identified by the class ace-presentation-info

by default, on mobile screen the information child moves bellow the main child, except if the parent has the class ace-presentation-hide-mobile-info

the information child usually has a different color schemes than the main one

add the class ace-presentation-container-reverse to the container to place the info container to the left

add the class ace-presentation-hidden-info to the container to hide the info panel when desired
