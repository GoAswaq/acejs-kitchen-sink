One of the types of panels predefined in the ace classes.

It is basically a vertical section of the screen, which modifies how some fields are displayed in it.

Ex. 1 - Quickbar

  • To define such a region, use the ace-quick-access-bar class.
  • It can be combined, like in this example, with an ace-advanced-panel class.
    • To show it, use the ace-show class.
    • To hide it, remove the ace-show class.
    • By default, the quickbar overrides the absolut positioning of the advanced panel. So order to force the panel to slide from the side, over the existing page content, add to it the class ace-not-default.
