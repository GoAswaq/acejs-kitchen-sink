This ace-col-classes can be used to set the width of ACE fields, ace grid columns, and normal html elements.

A subset of these classes may even be used to set the vertical height of elements. See Ex. 7.

Ex. 1 - ace-col Basic

  • ace-col-1 to ace-col-12 are a set if classes which set the width of an element.
  • This system devides the max available width in 12 segments of identical length.
  • ace-col-1 is the length of a segment, ace-col-2 is the length of 2 segments and so on until ace-col-12 which stretches the entire available width.
  • This classes on windows with the resolution bellow 680px all act as if they are ace-col-12: width = 100%
  • To test this, resize the browser window until the resolution requirement is reached.

Ex. 4 - ace-col Small

  • ace-col-m-1 to ace-col-m-12 are a set if classes which set the width of an element.
  • These classes take effect only on windows with the resolution between 681px and 1023px
  • ace-col-m-1 is the length of a segment, ace-col-m-2 is the length of 2 segments and so on until ace-col-m-12 which stretches the entire available width.
  • This classes on windows with the resolution bellow 680px all act as if they are ace-col-12: width = 100%
  • To test this, resize the browser window until the resolution requirement is reached.
  • In this example, on a normal desktop resolution the colored divs are sized with ace-col-6. But when the window is resized between 681-1023px, the ace-col-m- classes get activated.
