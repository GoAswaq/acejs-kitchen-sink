This tool helps easily build sections of the page which can expand and collapse.

Ex. 1 - Collapse

  • To define a collapsible element, use these 3 classes:
    • ace-collapsible-trigger: marks the trigger button.
    • ace-collapsible-content: marks the actual collapsible region .
    • ace-collapsible: marks the container in which the trigger button and the actual collapsible region.
  • The collapsible regions may be inbricated with one another.
  • The info buttons at the top of this application are made with this tool.
