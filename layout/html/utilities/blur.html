Blur Effects are used to hide, or show elements on the page using an effect of some kind.

The bluring is performed using a combination of classes and java script.

Ex. 1 - Bluring

  • The blured elements are marked with the class ace-blur-zone.
  • The blured elements are moved on the Y axis, bellow the height of their parent container.
  • To ensure that they are not visible, the parent should have for example it's overflow-y property set to none.
  • When they should be displayed, the class ace-show should be added to the blured elements. This will cause the elements to be animated into their visible position.
  • To hide them again, simply remove the ace-show class.

Ex. 2 - Deblur Zone

  • The blur zones may be combined with deblur zone, which could act as titles/header.
  • Deblur zones are marked with ace-deblur-zone class.
  • By default, they extent to 100% width, and have a distinct border
  • When the class ace-show is added to them, they constrict themselves
