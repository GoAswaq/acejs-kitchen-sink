Ex. 8 - Text modifiers

  • ace-white-space-normal: sets the white space property to normal; it uses !important
  • ace-no-text-wrap: prevents the text from going on the next line
  • ace-text-cut-elipsis: if the text does not fit the given element, ... will be added at the end of the text, to signify that there is more to it.
