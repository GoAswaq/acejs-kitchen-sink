These classes are used to set various distinct view styles on specific regions.

Ex. 1 - Entry Cards

  • An Entry Card is a container, that can colapse and expand itself to full screen.
  • To define an Entry Card, use the class: ace-entry-card.
  • To define a title bar region, use one of the classes: ace-entry-card-title or ace-title-bar.
  • To toggle full screen, use the class ace-full-screen.
  • To toggle collapsed state, use the class ace-colapsed.
  • Verious specific styles are applied to different element types.

Ex. 3 - Icon Cards

  • An Icon Card is used to display large icons, with a centralized text.
  • To create one, use the class ace-icon-card.
  • The modifier ace-icon-card-small can be added, to change their behaviour, and display a smaller icon.

Ex. 5 - Entry Panels

  • This type of panel can be used to format the information displayed in grid of type panel.
  • To create such a panel, use the ace-entry class.
  • It brings several styles to display fields, and images, and fa-many-arrows icon type.
  • Modifiers:
    • ace-active: it changes the style of the card
    • ace-entry-subtitle: marks the elements a subtitle

Ex. 6 - Checkable Cards

  • Checkable cards define areas that can be clicked.
  • How to create one:
    • Create a label, and give it the class ace-checkable-card.
    • Inside the label, add an input field of type checkbox.
    • Inside the label, add an empty span element, and give it the class ace-card-highlight.
