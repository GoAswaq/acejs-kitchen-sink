Various CSS classes that help with the setting the paddings and margins of various elements.

Ex. 1 - Paddings

  • The classes to manipulate the paddings of elements are split in the following sets:
    • Tiny, 5px: ace-tiny-padding-top,ace-tiny-padding-right,ace-tiny-padding-bottom,ace-tiny-padding-left,ace-tiny-padding
    • Small, 10px: ace-small-padding-top,ace-small-padding-right,ace-small-padding-bottom,ace-small-padding-left,ace-small-padding
    • Medium, 15px: ace-medium-padding-top,ace-medium-padding-right,ace-medium-padding-bottom,ace-medium-padding-left,ace-medium-padding
    • Big, 25 px: ace-big-padding-top,ace-big-padding-right,ace-big-padding-bottom,ace-big-padding-left,ace-big-padding
    • Large, 35 px: ace-large-padding-top,ace-large-padding-right,ace-large-padding-bottom,ace-large-padding-left,ace-large-padding
  • Attention: next to the above classes, there is another set of classes, which, instead of the ace- prefix, it uses the ace-i- prefix. This classes are created with the !important modifier.
  • Other padding control classes are: ace-no-padding, ace-no-padding-top, ace-no-padding-right, ace-no-padding-bottom, ace-no-padding-left, ace-form-padding.

Ex. 2 - Margins

  • The classes to manipulate the margins of elements are the following:
    • ace-small-margin-top,ace-small-margin-right,ace-small-margin-bottom,ace-small-margin-left,ace-small-margin
    • ace-medium-margin-top,ace-medium-margin-right,ace-medium-margin-bottom,ace-medium-margin-left,ace-medium-margin
    • ace-big-margin-top,ace-big-margin-right,ace-big-margin-bottom,ace-big-margin-left,ace-big-margin
    • ace-large-margin-top,ace-large-margin-right,ace-large-margin-bottom,ace-large-margin-left,ace-large-margin
  • Attention: next to the above classes, there is another set of classes, which, instead of the ace- prefix, it uses the ace-i- prefix. This classes are created with the !important modifier.
  • Other margin control classes are: ace-no-margin, ace-no-margin-top, ace-no-margin-right, ace-no-margin-bottom, ace-no-margin-left, ace-margin-auto.
