These are sets of classes that help to position elements, somehow on the page.

Classes that handle flex properties are not listed here. Please check the Flex section for futher details.

Ex. 1 - Float Classes

  • These classes are used to manipulate the floating settings.
  • The following classes have style written with the !important modifier, and they are used directly by the ace js code.
    • ace-left
    • ace-right
  • The following classes are not written without the !important modifier.
    • ace-left-float
    • ace-left-right

Ex. 2 - Positioning Classes

  • The following classes modify only the postion style of the an element:
    • ace-static-position
    • ace-relative-position
    • ace-fixed-position
    • ace-absolute-position
  • The following classes adjust but the position, and other attributes, to trigger certain results:
    • ace-to-bottom: moves the element to the bottom of the nearest positioned parent
  • Attention: in the examples bellow, the top, right, bottom, left controls are ignored for the following test classes: ace-to-bottom.
