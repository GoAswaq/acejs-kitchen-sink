This is a simple way to add a small icon in front of an element, ace, or otherwise.

Ex. 1 - Thumbs up

  • To add an icon, add the icon class, and the ace-thumbs-up class to the desired element.
  • In this example, the icon used is: fal fa-thumbs-up.
