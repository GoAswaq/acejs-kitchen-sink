This classes help set the height of elements.

Ex. 1 - Classes that set the height based on the full height of their parent, minus a fixed amount a pixels.

  • ace-full-height: set the height to 100%.
  • All other classes set the height to 100% - FIXED amount, where the fixed amount is the number specifeid in the class name:
    ace-height-minus-50,ace-height-minus-60,ace-height-minus-80,
    ace-height-minus-100,ace-height-minus-120,ace-height-minus-150,
    ace-height-minus-200,ace-height-minus-230,ace-height-minus-250,ace-height-minus-290,
    ace-height-minus-300,ace-height-minus-350,
    ace-height-minus-400,ace-height-minus-450
  • Attention: there is another set of classes, which instead of the ace- prefix are using the ace-i- prefix. This classes all use the !important modifier.

Ex. 3 - ace-vcol Vertical

  • ace-vcol-1 to ace-vcol-12 are a set if classes which set the height of an element.
  • This system devides the max available height in 12 segments of identical length.
  • ace-vcol-1 is the length of a segment, ace-vcol-2 is the length of 2 segments and so on until ace-vcol-12 which stretches the entire available height.
