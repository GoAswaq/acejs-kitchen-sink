Elements of tip datepicker are used to display and select a date.

By default, the date format used is: yy-mm-dd. It can be changed.

If the element is in a form, and it is porperly configured, the form will automatically retrive its value on the save operation, if the datepicker has been changed.

Ex. 1 - Simple Datepicker, default format: yy-mm-dd

  • When the label is aligned top and the datepicker has no value, the label will lower itsel, and play the role of placeholder.

Ex. 2 - Initial value, dd-mm-yy format, onchange event

  • In this example, the field has an initial value: 24-09-1982.
  • Also, the field is configured to work with a date format different than the default one: dd-mm-yy.
  • Each time a new date is selected, the an event is triggered: onchange, which can be intercepted.

Ex. 3 - onclose event

  • The onclose event is triggered each time the date selection window is closed, no matter if a new date was selected, or if the selection was canceled.
