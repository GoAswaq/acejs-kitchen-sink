Elements of tip checkbox are used to display an element, whose value may be changed between two states: on, and off.

By default, the on state is associated with 1, and the off state is associated with 0

The value for the two states may be customly defined.

Its functionality is extremely similar to that of the switch control. They differ from one another only visually.

If the element is in a form, and it is porperly configured, the form will automatically retrive its value on the save operation, if the checkbox has been changed.
