Elements of tip accordion help display a items that follow a given hierarchy.

Accordions do not have labels

Each element of the data displayed by the accordion needs to have at least 2 fields

  • A field with the name: name, which will be the text displayed.
  • A field with the name: tag, which will be the actual value of the option.
  • Attention: a tag should be unique in the data set of an accordion.

Whenver an option is selected, if the accordion has a handle function defined, the handle function will be called, and the tag of the selected option will be passed as a parameter.

Forms can set / get autmatically data from fields of this type.

Ex. 2 - Icons, children, initial tag

  • Some elements may have children. And those children may have children of their own.
  • When an element with children is first clicked, if the children are visible, they will be hidden. Otherwise, the children will be displayed.
  • An accordion may have an initial option selected. If this initial option is a child, then all its parents will be visible.
  • The accordion can be frozen, so that id doesn't expand or collapses when a parent option is clicked. Use Freeze / Unfreeze button to test this..

Ex. 3 - Checkboxes

  • Accordions may also have checkboxes.
  • Anytime a checkbox is checked or unchecked, an event is generated, which may be intercepted.
  • By default, the checkbox may be clicked ONLY if the checkbox is clicked. This may be changed, so that whenver an option is selected, it's checkbox is triggered as well, through the tooglecheckonrowclick setting. This is the case for this example.
  • By default, checking or unchecking a parent option will not have any affect on the check status of its children. This can be changed via the setting checkall. Set it to true, to enable this behaviour. This is the case for this example.
  • The accordion has two methods which can be used to retrieve the list of checked items:
    • getCheckedLeaves - which retrieves a list of all the items without children that are checked
    • getCheckedTags - which retrieves a list of all the items that are checked
