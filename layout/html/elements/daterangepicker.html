Elements of tip daterangepicker are used to display and select a date range.

By default, the date format used is: yy-mm-dd. Currently it cannot be changed (It the future it will).

The value returned on an *.ace('value')* will be a comma separated string, of the period start and end dates: YYYY-MM-DD,YYYY-MM-DD.

When the value is set on the field, the value must be formatted exactly as described above.
