Elements of tip cardview are when we want to easily hide an area, and replace it with antoher.

Ex 1 - Simple Example

  • By changing the selection, of the radio button, a different area will be displayed.
  • When creating a cardview element, these settings need to be specified:
    • contentselector: a CSS selector, which will identify which elements will be associated with the cardview; if it's not specified, the cardview will try to find all items with the class ace-card-view-item.
    • restricttoparent: if true, the selector will be applied only on the initial content of the cardview element, otherwise, it will be applied globally.
  • To switch to from a panel to another, the field's switchTo method should be used.
  • The desired panel can be identified in 2 ways:
    • by index ( 0 is the first index )
    • by selector
