Elements of tip menubutton are buttons, which when press will display a vertical menu, with several options

Whenver one of these options is pressed, something may happen.

Clicking anyway outside of the vertical menu closes the menu without triggering any other operation.

The elements displayed may have some of the fields

  • label - the text being displayed
  • value - a value which is passed to the cusom handler function when the label is selected from the vertical menu.
  • tag - a value which is passed to the cusom handler function when the label is selected from the vertical menu. Also, it is used for various other operations
  • type - the type of option displayed; by default it is a simple text, but it can also be: a radio button, a checkbox, a separator, group label or image. Simple items may have an icon displayed as well.

The menubutton has two other customizable element: an icon, and a text

  • In addition to the on onselect event that is triggered whenever a menu entry is selected ( with the expection of separators and group label), each element may also have a specific action defined.
  • Actions are custom functions that are called only for the elements for which they are specified
  • For this example, a general onselect method has not been defined, but the option 1 and 3 have an action specified
