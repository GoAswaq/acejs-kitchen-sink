Elements of tip prompt display a popup message to the user:

  • a simple message, an alert
  • a question, which may be answerd with yes / now, or ok / cancel, etc
  • a request for the user to type a text

While the popup is visible, the user may not interact with the rest of the GUI until the popup is dismissed
