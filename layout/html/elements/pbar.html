ACE fields of type progressbar are used to create progress bars.

The value of the field is a number betwen 0 and 100, and it represents how much the progress bar is loaded.

Ex. 1 - Example

  • Whenever the value of a progress bar changes, it will animate itself from its current position to its new value. To change its value, use the ACE value method.
  • The progress text on the bar may be displayed, or not. This is performed through the withtext configuration setting.
  • There are two modifier classes which may be applied on the progress bar:
    • ace-thin-progress-bar: makes the progress bar thinner.
    • ace-thinnest-progress-bar: makes the progress bar into a line; the text in this case is not visible.
  • The progress bar supports one user defined callback: onprogress.
  • To use it, a set of integer values betwen 0 and 100 needs to be specified. Each time the progress bar reaches one of thee defined values, if the onprogress method is defined, it will be called.
  • The interval set is specified as a comma separated string, or as an array through the progressinterval configuration setting.
  • In this example, each time a first progress bar reaches one of the following values: 0,10,20,30,40,50,60,70,80,100 its onprogress method will be called.
