Elements of tip radiogroup are used to select one option from a list of mutually exclusive options.

In this way, they might appear in function to radiogroupes, from which they differ in several major ways:

  • they look different, having all the posibile options visible at the same time
  • they have an edit mode build in: entries may be modified, or added
  • labels may be aligned only on top

Ex. 3 - Radiogroup, with data from Server + renderer

  • Initially, the radiogroup does not have any items.
  • Press the Load button, and the radiogroup will send am ajax request to the server, to retrive some data.
  • The data retrieved needs to have a specific format, encoded in a json.
  • Each loaded element has several fields, in addition to the name and id field.
  • Use getRecord to retrieve all of them.
  • A renderer function has been used which displayed additional stundent information.
  • Please note that a loading animation appears over the radiogroup for the duration of the server request.
